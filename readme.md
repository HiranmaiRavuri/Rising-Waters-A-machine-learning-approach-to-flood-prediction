# Rising Waters: A Machine Learning Approach to Flood Prediction

**Category:** Artificial Intelligence and Machine Learning

Rising Waters is a machine learning web application that predicts the probability of severe floods based on environmental and rainfall parameters. The system analyzes user inputs and instantly determines whether flood risk is high or low.

---

## Features
- Predicts flood risk in real time  
- Uses trained ML classification models  
- Clean web interface built with Flask  
- Fast and lightweight prediction system  

---

## Tech Stack
Python • Flask • Scikit-learn • XGBoost • NumPy • HTML • CSS

---

## Input Parameters
- Temperature  
- Humidity  
- Cloud Cover  
- Annual Rainfall  
- Jan–Feb Rainfall  
- Mar–May Rainfall  
- Jun–Sep Rainfall  
- Oct–Dec Rainfall  

---

## Output
- **Severe Flood Risk**
- **No Severe Flood Risk**

To demonstrate how machine learning can assist in early flood risk detection and support disaster preparedness.
